Software Products

Magistrala - Secure IoT Platform

Magistrala is a secure and open source IoT platform for device management, identity and access control, enabling reliable data flows across cloud and edge using MQTT and CoAP.

View Product
SuperMQ - MQTT Messaging Platform for IoT

SuperMQ is a scalable MQTT-based messaging platform for real-time IoT communication, supporting MQTT, CoAP, WebSocket and industrial IoT protocols.

View Product
Propeller - WebAssembly Orchestrator & IoT Rule Engine

Propeller is a serverless WebAssembly orchestration platform for secure IoT workloads, running across cloud, edge devices and microcontrollers.

View Product
mProxy - Multi-Protocol IoT Gateway

mProxy is a lightweight, scalable API gateway designed for the IoT edge. It enables seamless communication across MQTT, CoAP, HTTP, and WebSocket with real-time packet manipulation, pluggable authentication, and deep observability.

View Product
