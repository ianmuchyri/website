Rex

Rex is a lightweight rule engine for real-time event processing and decision automation across the Abstract Machines IoT ecosystem. It allows you to define declarative rules that act on streaming data, making IoT deployments smarter and more autonomous.

Download Rex

What is Rex

Rex enables declarative rule definition for intelligent, automated decision-making. It listens to message streams from SuperMQ and evaluates user-defined conditions to trigger actions.

Built to scale from edge to cloud, Rex can run inline with your IoT gateway or as part of a centralized deployment, giving you flexibility over where intelligence lives.

With a lightweight engine and YAML-based rule definitions, Rex empowers developers to build automation logic quickly and adaptively.

Rex Features

Stream-Based Processing

Rex listens to high-throughput event streams and evaluates rule conditions in real time.

Declarative Rules

Write simple, readable automation rules using YAML, no programming required.

Action Execution

Trigger commands, send messages, or activate workflows based on matched conditions.

Runs on the Edge

Deploy Rex next to your devices for ultra-low-latency logic at the network edge.

Highly Extensible

Easily extend Rex with custom actions or connectors tailored to your use case.

Open-Source and Lightweight

Rex is open-source under Apache 2.0 and optimized for low memory and CPU usage.
